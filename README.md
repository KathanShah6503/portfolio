# Kathan Shah Portfolio

A bold, recruiter-focused personal portfolio built with Next.js and TypeScript.

## Getting started

```bash
npm install
npm run dev
npm run lint
```

Then open [http://localhost:3000](http://localhost:3000).

## Pages

- `/` Home
- `/about`
- `/experience`
- `/projects`
- `/resume`

## Content source

Portfolio data is maintained in `data/site.ts`.
